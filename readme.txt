== Description ==

Функционал модуля:

- расчет стоимости доставки до ПВЗ и курьером;
- выбор ПВЗ в виджете при оформлении заказа и в админке для смены ПВЗ;
- применение настроек расчета и отображения ПВЗ из ЛК Boxberry;
- выгрузка заказов в ЛК Boxberry;
- формирование актов;
- получение ссылок на отслеживание заказов;
- печать этикеток и актов;
- смена статусов заказа в Woocommerce, после выгрузки в ЛК Boxberry;
- отображение текущего статуса заказа в Boxberry;
- добавление префиксов к номеру заказа для выгрузки в ЛК Boxberry

[Инструкция по установке и настройке](https://help.boxberry.ru/pages/viewpage.action?pageId=1704720)

[Техническая поддержка](https://sd.boxberry.ru)
